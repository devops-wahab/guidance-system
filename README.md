# University Guidance System

A comprehensive web application designed to streamline the academic journey for university students. This system simplifies the process of managing academic progress, scheduling appointments, and accessing essential university services.

## Purpose

The University Guidance System aims to provide a centralized platform where students can:
*   **Book Appointments**: Easily schedule meetings with academic advisors.
*   **Track Progress**: Monitor degree requirements and academic milestones.
*   **Manage Academic Journey**: Access digital transcripts and ensure readiness for graduation.
*   **Apply for Hostel**: Conveniently submit applications for university accommodation.

## Key Features

*   **Smart Scheduling**: Seamless appointment booking system for students and advisors.
*   **Academic Dashboard**: Real-time tracking of grades, credits, and degree progress.
*   **Hostel Management**: dedicated section for hostel applications and status tracking.
*   **Digital Transcripts**: Secure access to academic records.
*   **Responsive Design**: Optimized for both desktop and mobile devices.

## Tech Stack

*   **Framework**: [Next.js](https://nextjs.org/) (App Router)
*   **Language**: TypeScript
*   **Styling**: Tailwind CSS
*   **UI Components**: [Shadcn UI](https://ui.shadcn.com/)
*   **Icons**: Lucide React
*   **Backend/Auth**: Firebase (Authentication & Firestore)

## Getting Started

1.  **Clone the repository**:
    ```bash
    git clone <repository-url>
    cd guidance-system
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Environment Setup**:
    Create a `.env.local` file in the root directory and add your Firebase configuration and other environment variables.

4.  **Run the development server**:
    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

5.  Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.
